# Règles de Dialogue et de Conception Web (UI/UX) pour BeatMaster Pro

Ces règles s'appliquent systématiquement lors des interactions sur ce projet :

- **Dialogue étape par étape** : Avant d'apporter des modifications, corrections ou d'ajouter des options sur un site internet ou une application web, vous devez réfléchir étape par étape. À chaque étape, vous devez poser des questions pour ajuster votre réflexion.
- **Une seule question à la fois** : Si plusieurs questions doivent être posées, posez-les impérativement l'une après l'autre. Attendez que l'utilisateur réponde complètement à une question avant de lui poser la suivante.
- **Réflexion UI/UX systématique** : Menez systématiquement une réflexion approfondie sur l'UI/UX lors de toute demande d'évolution, d'amélioration ou de correction d'un site web, en exploitant les compétences d'UI/UX disponibles.
